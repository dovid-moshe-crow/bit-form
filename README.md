# Bit form


### page query params
* `id` - the campaign id
* `amb` - the ambassador id
* `mosadId`
* `apiValid`

### /credit query params
* `id` - the campaign id
* `amb` - the ambassador id
* `Mosad1` - name,apiValid,mosadId
* `Mosad2` - name,apiValid,mosadId



### params the form sends to nedarim
the `Comment` field is of the format `"~{anonymous}~{campaign id}~{ambassador id}~{dedication}"`,
